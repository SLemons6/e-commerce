# E-Commerce 

## About: 
This is a back-end for an internet retail web application that allows users to view, create, edit and delete products, as well as the categories and product tags that they belong to. 

## Built With: 
* JavaScript 
* Express.js 
* MySQL 
* Sequelize 
* Dotenv
